# AI Study Assistant

AI Study Assistant is an intelligent educational application built using Flutter and Groq AI to help students improve learning through smart AI-powered features.

## Live Website

https://aistudyassit.netlify.app/

---

## Overview

AI Study Assistant combines multiple learning tools into one platform. It helps students solve doubts, summarize PDF notes, generate quizzes, and interact using voice AI.

The application is designed to make studying faster, smarter, and more efficient.

---

## Features

### AI Chat Assistant
- Ask any academic doubts
- Get instant AI-powered responses
- Smart conversation history

### Voice AI Assistant
- Voice input support
- AI voice-based responses
- Hands-free learning experience

### PDF Notes Summarizer
- Upload PDF study materials
- Extract content
- Generate summarized notes instantly

### AI Quiz Generator
- Generate quizzes from notes
- Supports custom input
- Score calculation system

### Recent Activity Tracking
- Stores chat history
- Stores quiz history
- Stores PDF activities

### Theme Settings
- Light mode
- Dark mode

### Cloud Sync
- Future enhancement for online storage
